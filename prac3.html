Types of CSS

Inline CSS

Inline CSS is used to apply a unique style to a single HTML element. It is defined within the 'style' attribute of the HTML tag.

<p style="color:blue;">This is an example of inline CSS.</p>

Internal CSS

Internal CSS is used to define styles for a single HTML document. It is defined within a 'style' tag inside the 'head' section.

<style>
p {
color: red;
}
</style>

External CSS

External CSS is used to define styles for multiple HTML pages. It is defined in an external '.css' file and linked to the HTML document using the 'link' tag.

<link rel="stylesheet" type="text/css" href="styles.css">
